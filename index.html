Color Palette Generator

Usage

Palette size selects how many colors the palette will have.

Hue, Saturation, and Lightness are equation fields which accepts any valid JavaScript code, including regular numbers. Hue is measured in degrees, while Saturation and Lightness are measured in percentages.

Legend

  • Palette size: size
  • Color index (0 indexed): x
  • Normalized color index (Color index / (Palette size - 1)): i
  • Lightness: l
  • Chroma: c
  • Hue: h
